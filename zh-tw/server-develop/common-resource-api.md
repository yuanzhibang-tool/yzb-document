### 獲取用戶總數的接口

> 獲取使用的用戶總數的接口

**接口地址**

`POST /CommonResource/getUserCount`

**參數說明**

| 參數     | 是否必須 | 說明               |
| -------- | -------- | ------------------ |
| `app_id` | 是       | 開放平台應用的`id` |

**返回值**

成功的返回值

```json
{
  "status": "2000",
  "message": "ok",
  "data": {
    "user_count": "15"
  }
}
```
