## 通用资源相关接口说明

### 获取用户总数的接口

> 获取使用的用户总数的接口

**接口地址**

`POST /CommonResource/getUserCount`

**参数说明**

| 参数     | 是否必须 | 说明               |
| -------- | -------- | ------------------ |
| `app_id` | 是       | 开放平台应用的`id` |

**返回值**

成功的返回值

```json
{
  "status": "2000",
  "message": "ok",
  "data": {
    "user_count": "15"
  }
}
```
