> 用来快速实现应用在线配置的接口

---

### `config.get`

!>本接口自容器`1.3.0`版本之后提供

**主要描述**

| 主键          | 值                 |
| ------------- | ------------------ |
| **接口描述:** | 获取配置的`js`方法 |
| **备注**      | 无                 |

**参数验证说明**

| 参数  | 说明                                                  | 默认值 | 验证规则 |
| ----- | ----------------------------------------------------- | ------ | -------- |
| `key` | 如果不传则获取该用户下所有的配置,支持多个使用逗号分隔 | 无     | 无       |

**方法参数示例**

```json
{
  "key": "test-setting,123"
}
```

**方法回调`next`参数示例**

!>无论返回多个还是一个,都会按照该格式返回,没有则没有该值

```json
{
  "test-setting": "test-value",
  "123": {
    "key1": "v1"
  }
}
```
