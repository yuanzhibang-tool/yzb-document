_重要說明_

!>該 JS 庫需要在 header 中引入,可根據版本進行選擇
!>該庫使用 major.minor.patch 版本格式發布

#### 發布規則

| 版本類型 | 說明                                                                    |
| -------- | ----------------------------------------------------------------------- |
| `major`  | 主版本更新,包含部分接口調用破壞性更改,需要注意更新說明,以及版本升級指引 |
| `minor`  | 子版本更新,會包含部門接口的添加,不會影響現有接口的調用                  |
| `patch`  | 開發者無需關注,僅會涉及到普通問題修復和熱更新,不會影響接口調用          |
